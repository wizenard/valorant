## 能做什么

左边侧边栏列出全部特工，点哪个右边就显示哪个的技能。支持按职业筛选——决斗、先锋、控场、哨卫。搜索框输入特工名、英文名都能找到。

每个特工展示被动 + C/Q/E/X 四个技能，配上图标和文字说明。技能演示视频点击即可播放。

所有头像和技能图标都放在仓库里，不依赖外部图床。

---

## 怎么用

### 打开在线版

直接访问：**https://wizenard.github.io/valorant**

### 本地打开

```bash
git clone https://github.com/wizenard/valorant.git
# 进文件夹双击 agents_valorant.html 就行
```

### 启用 GitHub Pages

1. Fork 这个仓库
2. 进 Settings -> Pages
3. Source 选 `Deploy from a branch`
4. Branch 选 `main`，目录选 `/(root)`
5. 保存，等一两分钟就能访问了

---

## 仓库里有什么

| 文件 | 干嘛的 |
|------|--------|
| `agents_valorant.html` | 主页面 |
| `maps_assets/valorant_maps_data.js` | 地图数据 |
| `maps_assets/images/` | 地图缩略图和大图 |
| `特工技能/` | 特工头像、技能图标、背景图 |
| `.nojekyll` | 让 GitHub Pages 正常工作 |
